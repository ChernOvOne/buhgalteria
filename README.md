# Бухгалтерия — Операционная панель VPN-бизнеса

Система учёта финансов, клиентов и рекламных кампаний для VPN-сервисов.

## Быстрая установка

```bash
# Ubuntu 22.04 / 24.04
curl -sSL https://raw.githubusercontent.com/ChernOvOne/buhgalteria/main/install.sh | sudo bash
```

После установки: `buh` — CLI для управления.

## Возможности

- **Финансы:** доходы, расходы, баланс, категории, PDF/Excel отчёты
- **Клиенты:** Telegram ID, LTV, история платежей, retention/churn rate
- **Маркетинг:** UTM-воронка (клики → лиды → оплаты), ROI, CPA
- **LEADTEX:** готовые webhook для приёма лидов и платежей из TG-ботов
- **Платежи:** webhook API с автосозданием транзакций
- **Партнёры:** инвестиции, дивиденды, возврат инвестиций
- **Серверы:** мониторинг оплат VPN-серверов
- **Уведомления:** Telegram (платежи, конверсии, серверы)

## CLI

```bash
buh              # интерактивное меню
buh status       # статус сервисов
buh update       # обновление (с автобэкапом и health-check)
buh backup       # резервная копия БД
buh logs         # просмотр логов
```
