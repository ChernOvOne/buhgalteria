# Buhgalteria — Система учёта финансов бизнеса

## Быстрая установка на Ubuntu VPS

```bash
curl -sSL https://raw.githubusercontent.com/ChernOvOne/buhgalteria/main/install.sh | sudo bash
```

После установки войди: `http://YOUR_DOMAIN` — логин `admin`, пароль `admin123`

---

## Управление через CLI

После установки доступна команда `buh`:

```bash
buh              # интерактивное меню
buh status       # статус всех сервисов
buh start        # запустить
buh stop         # остановить
buh restart      # перезапустить
buh update       # обновить из git
buh backup       # резервная копия БД
buh restore      # восстановить из бэкапа
buh logs         # логи всех сервисов
buh logs backend # логи конкретного сервиса
buh info         # адреса и информация
```

## Роли пользователей

| Роль           | Права                                                       |
|----------------|-------------------------------------------------------------|
| Администратор  | Всё: CRUD всех данных, настройки, пользователи, категории   |
| Редактор       | Вносить и редактировать операции, видеть все данные          |
| Инвестор       | Только свои данные и общая статистика (только просмотр)     |
| Партнёр        | Только свои выплаты и доходная статистика (только просмотр) |

---

## Разделы системы

- **Дашборд** — KPI, графики, тепловая карта, партнёры, серверы, реклама, цели
- **Доходы** — ввод и история доходов с фильтрами
- **Расходы** — расходы по категориям, загрузка чеков
- **Реклама** — трекинг кампаний, ROI, стоимость ПДП
- **Сервера** — трекер хостингов с напоминаниями об оплате
- **Партнёры** — карточки с историей выплат и инвестиций
- **Инкас** — дивиденды и возврат инвестиций
- **Отчёты** — экспорт PDF и Excel за любой период
- **Настройки** — категории, пользователи, цели, Telegram, журнал аудита
